Technical Details

Positioning Algorithm:

// Major position: tile grid
gridX = margin + (tileX - minTileX) * baseTileSize + baseTileSize/2
gridZ = margin + (tileZ - minTileZ) * baseTileSize + baseTileSize/2

// Minor position: world coordinate offset within tile
offsetX = ((posX - minPosX_in_tile) / worldRangeX_in_tile - 0.5) * 2 * maxOffset
offsetZ = ((posZ - minPosZ_in_tile) / worldRangeZ_in_tile - 0.5) * 2 * maxOffset

// Final position
x = gridX + offsetX  // Centered on tile ¬± offset
z = gridZ + offsetZ

This ensures objects are centered around their tile grid position with fine offsets showing world coordinate precision.
